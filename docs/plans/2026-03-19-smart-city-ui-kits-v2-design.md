# Smart City + Urban-Adjacent UI Kits v2 Design

## Goal
Create ten production-worthy UI-kit docs sites for review: five Smart City directions and five urban-adjacent directions, each running in its own local git worktree.

## Shared Rules
- Every kit ships the same inventory contract: foundations, actions, forms, navigation, feedback, data, and composition.
- Use shadcn primitives where they exist, with domain-specific composition around them.
- No colorful gradients on white backgrounds.
- Distinctness must come from layout logic, component chrome, motion rhythm, and documentation IA, not only from color tokens.

## Smart City Directions
- Glass Control: dark civic glassmorphism, frosted telemetry, fast operational feedback.
- Stone Signal: restrained neumorphism, tactile mineral surfaces, accessible depth.
- District Bento: modular neighborhood service grid with warm neutral framing.
- Civic Current: crisp modern public-service product language in blue/slate.
- Quiet Bureau: typographic minimalism for public forms and service guidance.

## Urban-Adjacent Directions
- Mobility Studio: editorial transport system with directional energy.
- Climate Orbit: science-forward resilience reporting with orbital framing.
- Grid Forge: industrial orchestration with dense control surfaces and brutalist restraint.
- Care Mesh: humane clinical coordination with calm surfaces and guided forms.
- Campus Wave: contemporary education platform with collaborative modularity.

## Delivery Shape
- The base branch is the comparison hub and source of the inventory contract.
- Each worktree owns its own page architecture and local theme tokens.
- Review happens by running each worktree locally on its assigned port and checking contrast, focus, completeness, and distinctness.
